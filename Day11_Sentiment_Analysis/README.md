# 😊 What is Sentiment Analysis?

Sentiment Analysis is a Natural Language Processing (NLP) task that determines the emotional tone expressed in text.

A sentiment analysis model predicts whether a sentence is:

* Positive 😊
* Negative 😞
* Neutral 😐

It is one of the most widely used NLP applications in business and AI systems.

---

# Why is Sentiment Analysis Important?

Organizations use sentiment analysis to understand customer opinions and improve products and services.

Common applications include:

* Product review analysis
* Social media monitoring
* Customer feedback analysis
* Brand reputation tracking
* Movie review classification
* Chatbot emotion detection
