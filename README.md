🚀 30 Days of LLMs

> A 30-day hands-on learning journey exploring **Large Language Models (LLMs), Generative AI, Retrieval-Augmented Generation (RAG), AI Agents, and modern AI engineering** through practical experiments, code, and real-world projects.

📖 About

Welcome to my **30 Days of LLMs** challenge!

I'm Geetha Sri Singireddy, a Computer Science (AI & ML) student passionate about Generative AI, Large Language Models, AI Agents, and AI Research. This repository documents my daily learning journey as I explore the concepts behind modern AI systems through hands-on coding and experimentation.

The goal is not only to understand the theory but also to build practical implementations using industry-standard tools and frameworks.

🎯 Objectives

* Learn the fundamentals of Large Language Models
* Explore Prompt Engineering techniques
* Understand Tokenization and Embeddings
* Build Retrieval-Augmented Generation (RAG) pipelines
* Work with Vector Databases
* Experiment with Hugging Face models
* Learn LangChain and LangGraph
* Build AI Agents with tool calling and memory
* Explore local LLMs and model deployment
* Develop an end-to-end AI application

🛠️ Technologies & Tools
